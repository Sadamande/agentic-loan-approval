## 🔧 **TROUBLESHOOTING**

### **Problem: Double-click doesn't work**

**Linux:**
- Right-click file → Properties → Permissions → Check "Execute"
- Or run: `chmod +x /home/ubuntu/Desktop/demo/launch_app.sh`

**Windows:**
- Right-click → "Run as Administrator"

**Mac:**
- Right-click → "Open With" → "Terminal"

### **Problem: Python not found**

Install Python 3.8+ from https://www.python.org

### **Problem: Port already in use**

Close any other instances or edit the launcher script to use different ports.

### **Problem: Browser doesn't open**

Manually go to: http://localhost:8501
